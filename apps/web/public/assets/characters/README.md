Default pixel character assets for the frontend prototype.

## Generated assets

- `candidate-male-01.png` through `candidate-male-04.png`: generic male civic candidate presets.
- `candidate-female-01.png` through `candidate-female-04.png`: generic female civic candidate presets.
- `candidate-unknown.png`: generic unknown/undisclosed candidate silhouette.
- `data-principles-guide.png`: guide character for data principles and source notes.
- `default-civic-sprite-sheet-source.png`: generated chroma-key source sheet used to derive the transparent sprites.

These are generic UI placeholders and are not intended to resemble real people.
