# Data Policy

## 可用資料來源

僅使用官方或可信公開來源，例如：

- 中選會
- 監察院財產申報
- 監察院政治獻金
- 經濟部商工登記
- 公開資訊觀測站
- 政府電子採購網
- 司法院公開裁判書
- 地方議會公開資料
- 官方公告
- 可信新聞媒體

## 禁止事項

- 不使用需繞過權限的資料
- 不使用非公開取得資料
- 不登入非必要網站
- 不繞過 CAPTCHA 或 robots 限制
- 不提交 secrets

## 個資與風險限制

不得蒐集或輸出：

- 身分證字號
- 完整生日
- 完整住址
- 私人電話
- 私人 email
- 未成年子女姓名
- 未公開親屬資料
- 私人社群帳號
- 未經證實爆料

## 發布政策

- 未審核資料不得標記為 `verified`
- 未審核資料不得標記為 `is_public = true`
- D 級資料不可自動公開
- C 級資料建議人工確認後公開
